# Contributing to MultiCamApp

Thank you for your interest in improving MultiCamApp.

Contributions are welcome, including:

* Bug fixes
* Documentation improvements
* Camera compatibility testing
* UI/UX improvements
* Recording and verification improvements
* Platform-support investigation

Before submitting a pull request, please open an issue to discuss larger changes.

All contributions should follow the responsible-use principles of this project. MultiCamApp must not be used or modified for illegal surveillance, spying, unauthorized recording, harassment, privacy invasion, or any activity that violates the rights or safety of others.
