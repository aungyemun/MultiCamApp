Usage in MultiCamApp
--------------------

MultiCamApp uses ffprobe.exe solely as a standalone command-line tool
for post-recording video metadata inspection (Video Verification feature).

  - ffprobe.exe is invoked as a child process; no FFmpeg source code
    is compiled into or linked with MultiCamApp.
  - ffmpeg.exe is NOT bundled; only ffprobe.exe is included.
  - No modifications were made to the FFmpeg source code.
