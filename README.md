# MultiCamApp

MultiCamApp is a lightweight multi-camera recording app for synchronized video capture, metadata logging, and session verification.

The app is designed for controlled, lawful, and consent-based recording workflows. It was originally developed to support laboratory video recording, but it may also be useful for education, training, documentation, product testing, and other multi-angle recording needs.

## Current Status

MultiCamApp is currently under private development.

The Windows version is the primary target and is being tested with multiple USB camera layouts. macOS and Linux are not currently supported.

## Features

* 1–4 camera preview layouts
* Fixed-duration recording
* Multi-camera video capture
* Metadata logging
* Session-level verification
* Offline-first workflow
* Windows-first release plan

## Intended Use Cases

Possible use cases include:

* Research and experimental video recording
* Behavioral observation and locomotor activity recording
* Education, teaching, and practical demonstrations
* Technical training and skill documentation
* Multi-angle procedure or workflow documentation
* Product testing and quality-control recording
* Sports, movement, or posture observation
* Personal projects requiring synchronized multi-camera recording
